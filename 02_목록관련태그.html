ol (ordered list) : 순서 있는 목록

    type 속성 : 표시되는 숫자 모양을 변경
     type="a" : 소문자 a., b., c., d., ...
     type="A" : 대문자 A., B., C., D., ...
     type="i" : 로마 소문자 i, ii, iii, iv, ...
     type="I" : 로마 대문자 I, II, III, IV, ...

    start 속성 : 시작 숫자 지정
    reversed : 숫자 역순
    start="10" : 10부터 시작

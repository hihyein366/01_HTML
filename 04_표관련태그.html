표 관련 기본 태그

table(표)

    - 웹 문서에서 자료(데이터) 정리를 위해 사용하는 태그
    - table 태그는 행, 열을 작성하기 위한 판 나타냄

tr (table row)

    - 테이블의 행

td (table data)

    - 테이블의 열, 엑셀의 셀 하나

테이블 관련 추가 태그 1

th (table header) 

    - 테이블의 제목 or 내용 강조
    - 굵은 글씨, 중앙정렬 자동 적용

caption 태그

    - 테이블에 대한 설명, 머리말

테이블 관련 추가 태그 2

    [테이블 영역 지정 태그]

    thead : 테이블 상단 영역(컬럼 제목)
    tbody : 테이블 중간 영역(내용)
    tfoot : 테이블 하단 영역(합계, 이런거..)

    * 참고 사항
    table 태그 내부를 thead, tbody, tfoot으로 나누지 않으면
    모든 tr, td, th 태그는 tbody 내부에 작성된 것으로 취급.

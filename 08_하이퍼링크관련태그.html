a(ancohr : 연결, 고정) 태그

    - 현재 html 문서와 다른 html 문서를 연결시켜 페이지를 이동할 수 있게 만드는 거

    - a 태그로 감싸진 글/내용 == 하이퍼 텍스트(하이퍼 링크)

    - 하이퍼 텍스트는 다른 문서와 구분되는 html 만의 핵심 기능

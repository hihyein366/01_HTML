img 태그

    - 웹 문서에서 사진이나 그림 같은 이미지를 삽입하는 용도

    [속성] : 시작태그에 작성하는 내용

        1. src : 삽입할 이미지의 경로를 작성하는 속성
                 (파일 위치, 인터넷 주소)

        2. width / height : 너비 / 높이 지정

        3. alt(대안) : 사진 안나오면 나오는 글(설명 같은 거)
                       + 웹 접근성(스크린 리더가 이미지 설명 읽어줌)

img 예시

절대 경로로 이미지 출력하기

    절대 경로 : 변하지 않는 기준점으로 부터 목표까지의 경로 (C, D드라이브)
                (C:\workspace)

상대 경로로 이미지 출력하기

    상대 경로 : 현재 파일 / 페이지 위치를 기준으로 목표 까지의 경로 나타내는 것

고정 크기 단위(px) : 항상 고정된 크기

가변 크기 단위(%) : 화면 또는 영역에 대한 상대적인 크기 단위

고정 + 가변 크기

    고정 크기로 설정된 요소(태그) 내부에서 
    일정한 크기의 이미지 또는 요소를 출력하고 싶을 때
    가변크기로 지정
